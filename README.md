## Como instalar

* Para baixar o projeto, siga as instruções abaixo:

```
1. git clone https://github.com/erosMariano/findtrend.git
2. cd findtrend
```

* Instale as dependências e inicie o servidor:

```
3. yarn install
4. yarn dev
```

ou

```
3. npm install
4. npm dev
```
